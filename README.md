# Persona Mirror TG 🎭🔍
### Создавайте ИИ-контент, который невозможно отличить от оригинала

**Persona Mirror TG** — это продвинутый движок для анализа Telegram-каналов и извлечения "когнитивной ДНК" их авторов. Инструмент позволяет дистиллировать уникальный стиль, логику и внутреннюю мифологию любого канала в операционный гайд (`TG_STYLE_GUIDE.md`), который затем используется ИИ-агентами для автономного написания контента.

---

## 🚀 Почему это важно?

Большинство ИИ-инструментов просто копируют слова. Они анализируют ключевые слова и пытаются вставить их в текст, что часто приводит к результату, который "пахнет нейросетями". 

**Persona Mirror TG** идет глубже. Он анализирует не *что* пишет автор, а *из какого состояния* он пишет. Мы извлекаем "Цифровую душу" — систему убеждений, страхов и амбиций, которые формируют голос.

---

## 💎 Persona Mirror vs Обычный анализ

| Функция | Обычный ИИ / Промпт | Persona Mirror TG |
| :--- | :--- | :--- |
| **Анализ стиля** | Только частотные слова и Emoji | Психологический профиль (Core Drive) |
| **Структура** | Случайная или "литературная" | Специфический ритм автора (Ритм постов) |
| **Скрытый смысл** | Игнорируется | Теневая логика (подтексты и цели) |
| **Визуал** | Не анализируется | Анализ паттернов изображений и скриншотов |
| **Результат** | "Роботизированный" текст | Живой, узнаваемый голос автора |

## 🎯 Для кого этот инструмент?

- **Крипто-трейдеры и Альфа-хантеры**: Анализируйте экспертов и создавайте автономные каналы с качественным контентом.
- **Маркетологи и Бренд-менеджеры**: Создавайте ИИ-персон, которые общаются с аудиторией на языке вашего бренда.
- **Ghostwriters**: Автоматизируйте рутину, сохраняя уникальный стиль каждого клиента.
- **Владельцы сеток каналов**: Управляйте десятками каналов с уникальными голосами из одного центра.

---

## 🧱 Как это работает (5 простых этапов)

1. **Сбор материалов**: Инструмент анализирует 40-60 последних постов канала.
2. **Разбор на атомы**: Идентификация тона, лексики, синтаксиса и визуальных паттернов.
3. **Синтез "Цифровой Души"**: Определение "Основного импульса" (что движет автором) и "Теневой логики".
4. **Операционализация**: Генерация портабельного файла `TG_STYLE_GUIDE.md`.
5. **Автономность**: ИИ-агент использует гайд для написания постов, ответов и подбора контента.

## 📦 Структура репозитория

- `skill-tg-persona/` — основной скілл (модуль) для [OpenClaw](https://github.com/Ar3ss12/openclaw).
  - `SKILL.md` — зашифрованная логика и этапы процесса.
  - `prompts/` — системные промпты для Claude и GPT под этот скілл.
  - `tools/VISUAL_DNA.md` — методология анализа визуального стиля через Vision AI.
  - `references/TG_STYLE_TEMPLATE.md` — чистый шаблон в стиле "нотации для себя".
  - `references/vntralpha_example.md` — живой пример анализа (@vntralpha).

## 🛠 Быстрый старт

1. Скопируйте папку `skill-tg-persona` в ваш OpenClaw workspace.
2. Отправьте запрос агенту: `"Проанализируй тг канал @username"`.
3. Получите готовый гайд и начните генерировать: `"Напиши пост в стиле @username про биток по 100к"`.
