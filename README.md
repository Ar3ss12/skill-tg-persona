# Persona Mirror TG 🎭🔍

**Persona Mirror TG** — это продвинутый движок для анализа Telegram-каналов и извлечения "когнитивной ДНК" их авторов. Инструмент позволяет дистиллировать уникальный стиль, логику и внутреннюю мифологию любого канала в операционный гайд (`TG_STYLE_GUIDE.md`), который затем используется ИИ-агентами для автономного написания контента, неотличимого от оригинала.

## Основные возможности

- **Глубокий анализ (40-60 постов)**: Извлечение не только лексики, но и "Основного импульса" и "Теневой логики" автора.
- **Портабельный Style Guide**: На выходе вы получаете один файл, написанный от первого лица, который готов к использованию любым ИИ-агентом (OpenClaw и др.).
- **Анонимизация механик**: Движок использует профессиональную терминологию, скрывая сложные внутренние алгоритмы "дистилляции души".
- **Поддержка визуального ДНК**: Анализ паттернов использования изображений, графиков и мемов.

## Структура проекта

- `skill-tg-persona/` — основной скілл для OpenClaw.
  - `SKILL.md` — логика и этапы анализа.
  - `references/TG_STYLE_TEMPLATE.md` — чистый шаблон для ваших гайдов.
  - `references/vntralpha_example.md` — живой пример анализа реального Solana-канала.

## Как использовать

1. Установите скілл в ваш OpenClaw workspace.
2. Используйте триггер: `"Проанализируй тг канал @username"`.
3. Получите готовый `TG_STYLE_GUIDE.md`.
4. Используйте гайд для генерации постов: `"Напиши пост на тему X в стиле @username"`.
