# Contributing to Persona Mirror TG

We are excited about your interest in Persona Mirror TG! Whether you are an AI researcher, a crypto-native, or a marketing expert, your contributions can help make this tool even better.

## How to Contribute

### 1. Style Manifestos
If you have analyzed a public persona and created a high-quality `TG_STYLE_GUIDE.md`, feel free to submit it to the `examples/` directory. 
- Ensure the persona is public and the analysis follows our "Shadow Logic" principles.
- Use the provided `TG_STYLE_TEMPLATE.md`.

### 2. Logic Improvements
We are constantly refining the 5-phase analysis logic in `SKILL.md`. If you find a new pattern (e.g., specific punctuation usage or a new way to identify "rug-pullers" style), open a Pull Request.

### 3. Documentation
Help us make the README and guides even clearer. We value translations and "Quick Start" tutorials.

## Code of Conduct
- Be respectful to other contributors.
- Keep the focus on technical excellence and "Cognitive DNA" extraction.
- No spam or low-quality AI-generated PRs without manual verification.
